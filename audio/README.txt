The song is set inside the website's editor, not here.

Open the site → tap the faint dot (bottom-left) → passcode admin123 →
"Music" section → upload your MP3 and set the title/artist → Publish.

It commits here as song.mp3 and plays automatically when she opens the link
(it starts on her first tap/scroll — browsers block sound before that).

You can still drop a file named exactly  song.mp3  here manually if you prefer.
If no song is set, the music button just quietly disables itself.
